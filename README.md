# 科幻末日 ARPG (Sci-Fi Post-Apocalyptic ARPG)

這是一個使用網頁技術構建的 45 度視角 (Isometric) 動作角色扮演遊戲。

## 🛠️ 技術架構 (Tech Stack)

我們將使用以下工具來開發，這能確保您可以直接在瀏覽器中預覽遊戲，無需安裝額外軟體：

- **[Phaser 3](https://phaser.io/)**: 強大的 2D 遊戲引擎，非常適合處理 45 度視角的地圖和角色移動。
- **[TypeScript](https://www.typescriptlang.org/)**: 現代化的 JavaScript，讓程式碼更嚴謹，減少 Bug。
- **[Vite](https://vitejs.dev/)**: 極速的開發工具，讓您能即時看到修改後的成果。

## 🎮 遊戲特色 (Features)

- **視角**: 45 度上帝視角 (Isometric Projection)。
- **題材**: 科幻末日風格 (Sci-Fi / Post-Apocalyptic)。
- **操作**: 支援鍵盤與滑鼠控制角色探索地圖。
